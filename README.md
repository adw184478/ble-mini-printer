# ble-mini-printer
蓝牙热敏打印机

## 项目简介
本项目为小型蓝牙热敏打印机嵌入式控制方案，基于ESP32实现蓝牙串口数据接收、支持文本打印、缺纸检测、电量与温度监测告警，
采用环形缓冲区与状态机解耦数据接收与打印执行，运行稳定无阻塞。

## 硬件组成
- 主控：ESP32
- 通信：蓝牙
- 执行：热敏打印头、步进电机
- 输入：独立功能按键、纸张检测传感器
- 采集：电池电压采样、打印头温度采集
- 指示：LED状态灯（正常/低电/告警/打印启动）

## 软件架构
整体采用三层解耦设计，通过环形缓冲区与双状态机分离数据接收与打印执行，保障打印流畅度与系统实时性。

1. **外设输入层**：蓝牙接收回调、按键事件轮询、纸张检测中断、ADC周期性采样
2. **任务调度中心**：打印数据队列管理、设备与打印机双状态机、核心任务调度、定时器管理、打印控制逻辑
3. **执行输出层**：LED状态指示、步进电机驱动、打印头加热时序控制、SPI行数据传输、蓝牙状态告警上报
